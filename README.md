 # ⚡ RetroWave Downloader - MP4 Master Edition ⚡

**RetroWave Downloader** no es solo un gestor de descargas; es una cápsula del tiempo directa a los 80. Captura tus vídeos favoritos y flujos **HLS (.ts)** con una interfaz neón que haría llorar de emoción a los creadores de *Tron*.

## 🚀 Características "Turbo"

  * **Interfaz Synthwave**: Desarrollada con `CustomTkinter` para un look oscuro y eléctrico.
  * **Motor Dual de Descarga**:
      * **MP4 Directo**: Para enlaces directos de alta velocidad.
      * **HLS Assembler**: Detecta segmentos `.ts` y los cose automáticamente en un solo archivo MP4.
  * **Bypass de Seguridad**: Soporte para **Cookies de sesión** para evitar errores 403 (Prohibido).
  * **Auto-Centrado**: La ventana se posiciona matemáticamente en el centro de tu monitor al arrancar.
  * **Organización Inteligente**: Descarga directamente en tu carpeta oficial de **Vídeos**.

## 🛠️ Instalación y Uso

### Windows (Recomendado)

Descarga el instalador generado  .

> **Nota de Seguridad**: El instalador configura automáticamente las reglas del Firewall y exclusiones de Windows Defender para que la app pueda descargar sin bloqueos.

### Linux (Paquete .deb)

Si usas Ubuntu/Debian, puedes generar e instalar tu paquete:

```bash
chmod +x scripts/build_linux_deb.sh
./scripts/build_linux_deb.sh
sudo dpkg -i dist/retrowave-downloader_1.0.0_all.deb
```
